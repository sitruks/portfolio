Trivia Game
A simple trivia game featuring timers, where one can choose a time-based difficulty level. The game experience seeks to elicit the "beat the clock" sensation. The user can further customize the experience with input at the start menu. Whatever is input into this field will call the Giphy API and be dsiplayed as the result image at the end of the quiz.
